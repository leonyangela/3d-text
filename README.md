# 3D Text Playground

A simple **Three.js project** exploring 3D text, basic geometry, and camera perspective.  
The scene contains animated 3D text and torus (donut) shapes with a rotating camera to create subtle motion and depth.

Users can experiment with **camera FOV and Z-axis positioning** to see how perspective and distance affect objects in a 3D scene.

## Features
- **3D Text Rendering**          : Text generated using Three.js text geometry.
- **Animated Scene**             : Torus (donut) meshes and camera rotation add motion to the scene.
- **Camera Perspective Control** : Adjust the camera **field of view (FOV)** to change how the scene is perceived.
- **Depth Experimentation**      : Modify the **Z position** of the camera and text to observe depth and scaling effects.
- **Beginner-Friendly Project**  : Built to explore the fundamentals of Three.js and 3D scenes in the browser.
